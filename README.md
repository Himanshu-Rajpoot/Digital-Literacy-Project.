# Digital Literacy Project Report

## Overview
This project focuses on developing digital literacy skills, including safe internet usage, effective communication, and awareness of cyber threats. It covers practical tasks using various digital tools and platforms.

## Tasks Summary

### Task 1: Digital Literacy Awareness Infographic
Created an infographic using Canva to spread awareness about digital literacy, online safety, and responsible internet usage.

### Task 2: Digital Platforms Setup
Explored platforms like Google Drive and Gmail for communication, collaboration, and data storage.

### Task 3: Digital Skills Practice
Practised document creation, presentations, and basic coding using tools like MS Word and PowerPoint.
